Wrath of the Righteous Companion Detail

This API is a resource for the companions in the game Pathfinder: Wrath of the Righteous. The routes are as follows:
Route Method Description Example 
/companions
 GET Gets a list of companions' names, ids, classes, and the chapter you can first recruit them. List companions 
/companions/:id
 GET Returns the detail about the companion matching the id in the parameter Get companion 1
